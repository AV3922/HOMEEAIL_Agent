API simulation + adaptive decision agent + CSV/XLSX ingestion + documentation in one HTML file.

Connect any device/system by defining a protocol profile. Agent can consume data and issue actions/recommendations.

2) Hardware Data Upload (CSV/XLSX)

Expected columns: timestamp,deviceId,powerKw,temperature,pricePerKwh,occupancy

3) HyperAgent Decision Engine

Runs autonomously on ingested telemetry + learned policy weights. Produces actions and user recommendations.
